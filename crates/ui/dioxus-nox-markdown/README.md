# dioxus-nox

> **⚠️ Disclaimer:** This entire library was generated by AI (Claude) as part of a
> personal learning project. It has not been battle-tested in production, has not
> undergone formal security review, and may contain subtle bugs or unsound
> abstractions. Use it at your own risk and exercise extreme caution before
> depending on it in anything that matters.

Headless component library for [Dioxus 0.7](https://dioxuslabs.com/) (Rust/WASM), following the [Radix Primitives](https://www.radix-ui.com/primitives) pattern.
